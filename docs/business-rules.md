# Regras de negócio ANA + Feegow

## Atendimento

- Canal oficial: WhatsApp.
- Plano: particular.
- Profissional inicial: Dra. Thalita.
- Local: único e fixo.
- Usuário Feegow: ANA.

## Agendamento

- A ANA nunca inventa horários.
- A ANA sempre consulta o Feegow antes de oferecer agenda.
- A reserva deve ser criada antes do pagamento.
- A consulta só é confirmada após pagamento.

## Retorno

- Retorno é permitido até 30 dias após a consulta anterior.
- Após 30 dias, o atendimento deve ser tratado como nova consulta paga.
- O MCP deve validar essa regra antes de criar agendamento como retorno.
